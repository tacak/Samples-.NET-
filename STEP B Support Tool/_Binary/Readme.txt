STEP B の文章整形をサポートするツールです。


必要環境
========
フロント ( GUI ) が .NET Framework 2.0 ( C# ) で、VC++ 2008 で作った 
DLL を呼び出している形なので、.NET Framework 2.0 のランタイムと、
VC++ 2008 のランタイムが両方必要です。ご注意ください。

加えて、一時ファイルを作るため、読み取り専用のパス ( CD-ROM など ) から
直接実行はできませんので、あしからず。

ランタイムとか何のことだか分からんという方は、セットアップ作っておいた
ので、それを利用してインストールしてください。それで、ランタイムを含め
て、必要なファイルがインストールされます。( setup.exe を実行してくださ
い )


使い方
======
テキストエリアに整形対象の文章を貼り付けて、「STEP B 整形」ボタンを押す
だけです。

「整形が終わったら、クリップボードにコピー」のチェックを ON にしている
と、整形終了と同時に、整形後の文章をクリップボードにコピーします。

整形に失敗したと思った場合は、整形前に戻すボタンを押すことで、整形前の
状態に戻すことができます。


機能説明
========
1. ダメな文字の変換機能
以下のような変換を行います。
・全角英数字と全角記号は、半角に変換します。
・半角カナは、全角に変換します。
・一桁の丸付き数字は、(1) などに変換します。


2. 半角スペース挿入機能
全角文字と半角文字の間に半角スペースを挿入します。
ただし、"、" の前後には半角スペースを挿入しません。


3. 改行削除機能
以下の条件全てに合致する改行を削除します。
・行単位で見て、指定バイト ( GUI で設定可能 ) 以上の位置にある改行
・改行の前の文字が "。",")",">",":", ";","---","--+","===","###","***" のいずれでもない


4. 改行補完機能
"。" を発見した場合、改行を挿入します。
整形前から、"。" の後ろに改行がある場合は、挿入しません。


5. 辞書置換機能
辞書ファイル ( dictionary.txt ) を基に、製品名の修正を行います。
辞書ファイルの書き方は、辞書ファイルの書き方.txt をご参照ください。

なお、大文字・小文字間違い ( windows -> Windows など ) は、大文字・小文
字が正しい名称を辞書に登録しておけば、自動的に修正します。

6. SWH / GWN のアドレス補完機能
SWH や GWN の公開 QA および Tips のアドレスを発見した場合、公開用に 
SWH と GWN のアドレスを二つ並べます。


7. 行頭 URL の括弧補完機能
行頭から、URL が始まっている場合に限り、< > を補完します。

変換前 :
・グーグル
http://www.google.co.jp/

変換後 :
・グーグル
< http://www.google.co.jp/ >

こういう風に、文章の途中に http://www.google.co.jp/ などがあっても、
括弧の補完は行いません。


8. 整形除外機能
行頭に /@@ がある行から、行頭に @@/ がある行までは整形を行いません。
行単位の処理であることに注意してください。行単位で見て、これらの整形除
外記号の前後に文字があってはいけません。

駄目な例 :
/@@ 整形除外文章 @@/

良い例 :
/@@
整形除外文章
@@/

なお、F5 で /@@ を F6 で @@/ をカーソル位置に挿入する補助機能がありま
す。
